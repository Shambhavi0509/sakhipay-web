📝 How to File a Complaint (Cybercrime)

  • Go to cybercrime.gov.in
  • Select “Report Cyber Crime” and click "Report Other Cyber Crimes"
  • Register/Login using your mobile number
  • Fill in complaint details: incident date, platform (WhatsApp, UPI, SMS), phone number used, etc.
  • Attach any screenshots, bank details (if needed)
  • Submit and save complaint ID for tracking
